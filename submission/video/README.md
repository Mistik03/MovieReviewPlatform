# Video — what to record

The guidelines require a short video showcasing the final application in action. Record it
when the cloud deployment is live (or against localhost) and place the file in this folder
before packaging.

Suggested tool: Windows Game Bar (Win+Alt+R) or OBS Studio. 1080p, 2–4 minutes is plenty.

## Suggested storyboard (mirrors the demo slide)

1. Home page — let the hero animation play, scroll through the poster rows. (~20 s)
2. Browse — filter Films/Series, pick a genre chip, search for a title, change sort. (~30 s)
3. Title detail — scroll the cast, show the average rating. (~20 s)
4. Register a new account, then rate (1–10 bar) and review the title. (~40 s)
5. Try posting a second review — show the "already reviewed" error (business rule). (~10 s)
6. Log in as admin — Admin → TMDB Import: search a movie, import it, open its new page. (~40 s)
7. Swagger — expand an endpoint, show the JWT Authorize button, call one endpoint. (~20 s)

Name the file something like `marquee-demo.mp4`.
