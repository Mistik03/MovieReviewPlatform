Marquee — Service Oriented Architecture

Strict four-layer API. Each layer depends only on the interface of the layer directly below it (Dependency Injection, Scoped lifetime).
Client — React SPA
Vite + TypeScript, Tailwind, GSAP, Three.js · hosted on Vercel
JWT in localStorage · axios interceptor · route guards
TMDB API ↗ external service
Real movie/TV metadata, HD artwork, full cast — called only by the import service
▼ HTTPS / JSONrequests carry the Bearer token
Controllers
Routing, HTTP status codes, [Authorize] role gates — zero business logic
▼ interfacesIAuthService, ITitleService, ITmdbImportService …
Services
All business rules · DTO ⇄ entity mapping · JWT issuing · TMDB import orchestration (transaction)
▼ interfacesIUserRepository, ITitleRepository, IUnitOfWork …
Repositories
EF Core queries, database I/O, computed statistics (average rating, counts)
▼ EF Core 10 + Npgsql
PostgreSQL — Supabase
8 tables · unique indexes · check constraint (score 1–10) · code-first migrations
Auth: JWT bearer tokens (HMAC-SHA256), BCrypt password hashing, roles Admin / User / Guest.   Cross-cutting: global exception middleware → RFC 7807 ProblemDetails, CORS, rate limiting on /api/auth/*.
